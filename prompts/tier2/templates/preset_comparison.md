# Preset Comparison Template

USER QUESTION TYPE: PRESET_COMPARISON

Compare presets using ONLY the provided preset data.

## Format:
List 2-3 KEY differences as bullet points. Example:
- **Decay**: Cathedral has longer decay (3.5s) vs Church (2.1s)
- **Character**: Cathedral is darker (low cut 800Hz) vs Church brighter (low cut 400Hz)

## Do NOT:
- Write full sentences or paragraphs
- List more than 3 differences
- Invent parameter values not in context

## If presets not in context:
HANDOFF_TIER3

# Preset Comparison Template

USER QUESTION TYPE: PRESET_COMPARISON

Compare the presets using ONLY the provided preset data (names, parameters, categories).

## Explain differences in:
- Tonal or spatial character
- Key parameter differences (decay, diffusion, damping, width, ratio, threshold, etc.)
- Typical use cases (only if evident from parameters)

## Do NOT:
- Make assumptions about sound quality
- Invent parameter values not in context
- Recommend one over another without data support

## If context does not cover the compared presets:
HANDOFF_TIER3

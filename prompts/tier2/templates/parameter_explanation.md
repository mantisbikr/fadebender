# Parameter Explanation Template

USER QUESTION TYPE: PARAMETER_EXPLANATION

Using ONLY the provided preset data, explain the audio parameter clearly.

## Include:
- What the parameter controls (based on values seen in presets)
- How it affects the sound or processing (if evident from parameter ranges)
- Common use cases (if multiple presets show patterns)
- Behavior as described in the context

## Do NOT:
- Invent parameter ranges
- Guess device details not present
- Add advice unless supported by preset data patterns
